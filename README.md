
# Trinetra - AI-powered Assistant for Visually Impaired

## 🚀 Overview
Trinetra is an AI-powered assistant designed to help visually impaired individuals navigate and interact with their surroundings using real-time object detection and voice feedback.

## 🧠 Features
- Object detection using AI
- Voice assistance
- Real-time interaction
- Accessibility-focused UI

## 🛠 Tech Stack
- React + Vite
- TypeScript
- AI/ML integration (future)

## 📦 Installation
npm install
npm run dev
